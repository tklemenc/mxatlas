Email Providers of German Municipalities

What is this?

A map of German municipalities showing which provider class handles their official email, based on public DNS records.

Context

Digital sovereignty: municipal email infrastructure reveals dependence on hyperscalers, public IT service providers, commercial hosters and shared relay setups. This map makes the current landscape visible.

How does it work?

Each municipality's official domain is checked via public DNS records and mapped to an indicative provider class.

This is a static snapshot based on publicly visible DNS indicators, not an official statement by the municipality.

Open source & open data

The code and data are on GitHub.
If you notice an error, please open an issue with the municipality key and the expected provider class.

Disclaimer

Results are indicative, may be incomplete or outdated, and should be read as technical hints derived from public DNS records.
